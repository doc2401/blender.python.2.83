bpy.ops.mball.select_all(action='TOGGLE')¶

Change selection of all meta elements

Parameters

action (enum in ['TOGGLE', 'SELECT', 'DESELECT', 'INVERT'], (optional)) –

Action, Selection action to execute

  • TOGGLE Toggle, Toggle selection for all elements.

  • SELECT Select, Select all elements.

  • DESELECT Deselect, Deselect all elements.

  • INVERT Invert, Invert selection of all elements.

bpy.ops.mball.select_random_metaelems(percent=50.0, seed=0, action='SELECT')¶

Randomly select metaelements

Parameters

  • percent (float in [0, 100], (optional)) – Percent, Percentage of objects to select randomly

  • seed (int in [0, inf], (optional)) – Random Seed, Seed for the random number generator

  • action (enum in ['SELECT', 'DESELECT'], (optional)) –

    Action, Selection action to execute

    • SELECT Select, Select all elements.

    • DESELECT Deselect, Deselect all elements.
